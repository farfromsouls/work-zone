# Relax & Focus

A free desktop application for relaxing with ambient music/sounds, managing tasks, and boosting productivity with a built-in timer.

## Installation

### Option 1: Direct Run (Recommended for Windows)
1. Download the application files
2. Run `createShortcut.bat`
3. A shortcut will be created â€“ use it to launch the app

### Option 2: Manual Setup
1. Ensure Python 3.10 is installed
2. Create a virtual environment: `python -m venv venv`
3. Activate the environment and install requirements:
   ```bash
   pip install tkinter pygame
   ```
4. Run the application: `python main.pyw`

## Requirements
- Python 3.10
- tkinter
- pygame

## **!!! WARNING !!!**
Python versions higher than 3.10 - won't work because of problems with sound libraries.
